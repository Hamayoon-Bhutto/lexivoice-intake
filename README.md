 # LexiVoice Intake

LexiVoice Intake is a portfolio demo project for an AI-powered legal client intake system.

It helps law firms automate inbound client intake using:
- AI voice intake
- Legal lead qualification
- Case urgency detection
- Document checklist generation
- CRM-style dashboard
- Call transcript and summary
- Simulated email/SMS follow-ups

## Tech Stack

- FastAPI
- LangGraph
- Supabase
- Next.js
- Tailwind CSS
- Retell AI / Twilio trial
